# Rose-App Project Context & Documentation

## 1. Project Overview
- **Name:** Rose-App
- **Tech Stack:** Next.js (App Router), TypeScript, Tailwind CSS, NextAuth, i18n.
- **Architecture:** Feature-based (`src/features`, `src/shared`).

## 2. Project History (Summary)
- **Sprint 1 (Foundation):** Setup, Design System (Colors, Typography, Dark/Light), UI Kit (Buttons, Inputs, etc.), i18n routing.
- **Sprint 2 (Auth Flow):** Implementation of Login, Register (Multi-step), Forgot Password (Reset flow), NextAuth, and Middleware.
- **Git State:** Repository history was cleaned/reset to ensure atomic, task-based commits (feature branches).

## 3. Current Focus (Sprint 3)
- **Goal:** Homepage UI implementation.
- **Tasks:** Hero/Features, Best Selling, Popular Sections, About/Gallery/Partners, Header/Footer.

## 4. Coding Standards
- Strict TypeScript usage.
- Feature-based structure.
- Clean Code & Atomic Commits.
- Use `src/shared` for common utilities/components.

## 5. Development Roadmap
- We are currently transitioning from Auth logic to UI/Content implementation.
